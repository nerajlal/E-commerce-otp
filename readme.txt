Mini eCommerce Website with OTP Login

Project Structure:
 /includes/    -   Common PHP includes (DB connection, header, footer)
 /auth/        -   User login logout
 /admin/       -   Admin panel to manage products
 /assets/      -   Product images
 /DB/          -   Database file
 index.php     -   Homepage
 cart.php      -   User cart
 checkout.php  -   Place order


Technologies Used:
 Core PHP
 MySQL
 Bootstrap 5


Setup Instructions:
1. Install WAMP + MySQL setup
2. Place project folder inside `www` directory
3. Open phpMyAdmin:
    Create a database named: `ecommerce`
    Import the sql file
4. Start Apache and MySQL
5. Open browser and go to: http://localhost/Ecommerce_otp/


Admin Login:
